# Cyber Realm

Cyber Realm is the public game-design repository for the Cyber Realm trading card game and its training-focused cybersecurity editions.

The first active edition is **ICS-WATER**, a water-sector ICS/OT tabletop training edition built around Red Team vs Blue Team gameplay.

## Core goals

- Maintain card names, mechanics, and decklists as structured files.
- Develop balanced card text, flavor text, and learning notes.
- Support multiple Cyber Realm editions from one repo.
- Track playtesting, mechanics, and training objectives.

## Realms

Cyber Realm uses six core realms:

- Cloud
- Internet
- Network
- Application
- Endpoint
- Infrastructure

## Card types

- Realm
- Asset
- Attack
- Defense
- Tool
- Protocol
- Operator
- Event
- Legacy

## Current edition

```text
data/editions/ICS-WATER/
```

ICS-WATER is designed for AWWA-style tabletop training where players learn how cyber attacks affect water operations and how defenders detect, respond, recover, and reinforce.

## Public workflow

```text
Card concepts
  -> card database
  -> rules text
  -> learning notes
  -> playtest decklists
  -> balance revisions
```

Art generation, design layout, print preparation, and production workflow are handled outside this public repository.

## First playtest target

**Small Utility vs Script Kiddie**

This first playtest deck focuses on beginner-friendly cards, legacy systems, remote access, patching, network segmentation, and operational recovery.
