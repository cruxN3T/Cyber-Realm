# Script Kiddie Red Team Balance Note

## Question

The Script Kiddie Red Team has fewer unique card names than the Small Org Blue Team. Does that hurt balance?

## Answer

No. Fewer unique Red Team cards does not hurt the deck as long as the Red Team has enough duplicate pressure cards and a clear attack chain.

For Tier 1 training play, fewer unique attacker cards can actually help because the Red Team becomes easier to teach and easier for AWWA players to understand.

## Why this works

The Red Team deck is not weaker just because it has fewer unique names. It has repeated cards that create consistent pressure:

- Network Scan x2
- Public Info Gathering x2
- Downloaded Exploit x2
- Malicious Link x2
- Malicious Attachment x2
- Vulnerability Exploit x2
- Credential Attack x2
- Password Spray x2
- Operations Budget x4
- Network Position x2
- External Access x2
- App Access x2

This makes the Script Kiddie deck feel realistic. Low-sophistication attackers repeat the same tactics: scanning, public info gathering, phishing, credential attacks, copied exploits, and basic malware.

## Training value

For AWWA and water-sector training, repeated attacker cards are useful because players see the same concepts more than once. That improves learning.

A beginner player should not have to learn 32 completely different attacker cards in the first session.

## Balance risk

The main balance risk is not the number of unique cards. The main risk is whether Red Team draws enough resources and attack cards in the first 2-3 turns.

If Red Team feels too weak during playtesting, adjust by adding one more duplicate of a core pressure card, not by adding many new unique cards.

Recommended Red Team buff options, in order:

1. Add one more External Access or App Access.
2. Add one more Network Scan.
3. Add one more Credential Attack or Vulnerability Exploit.
4. Reduce Command Channel cost from 2 to 1.

## Current verdict

The current Script Kiddie deck is likely balanced enough for the first Tier 1 prototype.

Fewer unique cards helps onboarding, supports repeated learning, and makes the attack pattern feel realistic.

Do not add more unique Red Team cards yet. Playtest first.
