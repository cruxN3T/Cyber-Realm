# Realm Resource / Funding Cards

## Design decision

Capital Funding should not be the only upgraded resource card. Because the Tier 1 ICS-WATER decks include Network, Application, Internet, Endpoint, and Infrastructure cards, each important realm should have a clear resource/funding card option.

## Resource card types

Use two resource levels:

1. **Basic resource cards** — common, repeatable, used in training boxes.
2. **Funding / upgrade resource cards** — stronger resource cards that help play assets, defenses, tools, or operators in a specific realm.

## Training box rule

All Training Box cards should be Common only.

If a card is built for the Training Box, keep the mechanic simple and avoid rarity-based power spikes.

## Recommended Tier 1 resource cards

| Card Name | Category | Realm | Color | Cost | Text | Training Role |
|---|---|---|---|---:|---|---|
| Operations Budget | Realm | Infrastructure | Green | 0 | Provides 1 Infrastructure resource. | Baseline utility operations funding. |
| Network Maintenance Budget | Realm | Network | Black | 0 | Provides 1 Network resource. | Supports firewalls, port closure, scanning, and network defense. |
| Application Security Budget | Realm | Application | White/Grey | 0 | Provides 1 Application resource. | Supports MFA, SCADA web interface, credential controls, and app-layer defense. |
| Internet Access Budget | Realm | Internet | Blue | 0 | Provides 1 Internet resource. | Supports internet-facing access, remote connectivity, and external exposure management. |
| Endpoint Support Budget | Realm | Endpoint | Red | 0 | Provides 1 Endpoint resource. | Supports laptops, antivirus, password reset, and endpoint recovery. |
| Capital Funding | Realm | Infrastructure | Green | 0 | Provides 1 Infrastructure resource. If used for an Asset, that Asset enters with +1 Defense. | Upgrade funding for physical/operational assets. |

## Optional upgraded funding cards

These can be used later in Battle Decks, Starter Decks, Expansion Packs, Collector Boxes, or selected Booster Packs.

| Card Name | Category | Realm | Color | Suggested Text |
|---|---|---|---|---|
| Network Upgrade Funding | Realm | Network | Black | Provides 1 Network resource. If used for a Network Defense or Tool, reduce incoming Network Attack damage by 1 this turn. |
| Application Hardening Grant | Realm | Application | White/Grey | Provides 1 Application resource. If used for MFA or a credential defense, draw 1 card then discard 1 card. |
| Internet Exposure Reduction Grant | Realm | Internet | Blue | Provides 1 Internet resource. If used for Port Closure or Gateway defense, cancel one Recon bonus. |
| Endpoint Refresh Budget | Realm | Endpoint | Red | Provides 1 Endpoint resource. If used for Operator Laptop, Antivirus, or Password Reset, restore 1 Defense to an Endpoint Asset. |

## Recommendation for the first Tier 1 Training Box

For the first 64-card prototype, do not overload the deck with too many special funding cards.

Use simple Common resource cards so players understand cost and realms quickly:

- Operations Budget — Infrastructure
- Network Maintenance Budget — Network
- Application Security Budget — Application
- Internet Access Budget — Internet
- Endpoint Support Budget — Endpoint

Keep Capital Funding as the Infrastructure upgrade card if the deck needs a stronger utility-funding lesson.

## Why this matters

If Blue Team has cards in multiple realms but only Infrastructure resource cards, players may get stuck with cards they cannot play. Adding realm-specific resource cards improves gameplay flow and teaches that different areas of a utility require different types of investment.
