# First Playtest Readiness

Before the first guided test, confirm:

- [ ] All 64 cards exist in prototype form
- [ ] Duplicate cards have the right copy counts
- [ ] Blue Team has 32 cards
- [ ] Red Team has 32 cards
- [ ] Card types follow the category rules
- [ ] Blue Team has Defense cards, not Attack cards
- [ ] Red Team has Attack cards, not Defense cards
- [ ] Lifecycle symbols are visible enough to support training
- [ ] Realm colors are visible enough to support fast recognition
- [ ] Card text is readable at card size
- [ ] No proprietary production workflow details are committed

---

# First Playtest Questions

Ask after the first test:

- Was the goal clear within five minutes?
- Did the Red Team attack chain feel realistic?
- Did the Blue Team defense lifecycle feel useful?
- Which card was the most fun?
- Which card was confusing?
- Which card felt too strong?
- Which card felt too weak?
- Did water-sector players recognize the scenario?
- Did security players recognize the attack and defense logic?
- Would this work as a 2-hour AWWA tabletop session?
