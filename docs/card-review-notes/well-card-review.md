# Card Review: Well

## Current card

- Name: Well
- Category/role line: Asset - Source Water (AWIA-S)
- Realm: Infrastructure
- Color: Green
- Game text: Generates 1 Infrastructure resource each turn. If destroyed, Blue Team loses 1 SCP.
- Flavor text: The smallest source can become the most important asset.
- Value: 0/3
- Connection: RTU/PLC

## Review

This card works well as a Tier 1 Small Org asset.

The card correctly treats the Well as an operational asset, not as a lifecycle action card. The Infrastructure realm symbol in the upper-right is the right main symbol to use here.

## Recommended interpretation

Do not treat the Well as a Reinforce action. Treat it as a protected operational asset that supports the defense economy.

The card teaches:

- Source water matters.
- Small utilities still have critical cyber-physical dependencies.
- A cyber event can cause operational impact if it affects connected control paths.
- RTU/PLC dependency matters.

## Suggested final wording

Generates 1 Infrastructure resource each turn. If destroyed, Blue Team loses 1 SCP.

Flavor:

The smallest source can become the most important asset.

Connection:

RTU/PLC

## Optional tiny improvement

If space allows, consider changing the category line to:

Asset - Source Water / Critical Asset (AWIA-S)

This makes the card easier for non-security water-sector players to understand.

## Design note

The 0/3 value is good. Assets having 0 attack and 3 defense is simple and easy to teach.
