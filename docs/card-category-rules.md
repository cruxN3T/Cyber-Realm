# Card Category Rules

Every Cyber Realm card must use one of these card categories:

- Asset
- Defense
- Attack
- Tool
- Protocol
- Operator
- Realm
- Event

## Important distinction

Terms like Recon, Credential Attack, Remote Access Attack, Malware Attack, Recovery, or Scenario are roles, tags, keywords, or playtest notes. They are not card categories.

For example:

- Port Scan = Attack
- Password Spray = Attack
- Basic MFA = Defense
- Operations Budget = Realm
- Field Technician = Operator
- Alert Fatigue = Event
- Modbus TCP = Protocol
- Deep Packet ICS Monitor = Tool
- Water Tower = Asset

## Public repo rule

Public card files should keep categories simple and consistent. Private art generation, Canva work, and print production details stay outside this repository.
