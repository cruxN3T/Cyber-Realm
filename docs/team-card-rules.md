# Team Card Rules

Cyber Realm uses different combat card categories for each side.

## Core rule

- Blue Team uses **Defense** cards.
- Red Team uses **Attack** cards.

## Allowed team/category mapping

| Team | Allowed combat category |
|---|---|
| Blue Team / Defender | Defense |
| Red Team / Attacker | Attack |

## Shared card categories

Both sides may use shared support categories when appropriate:

- Asset
- Tool
- Protocol
- Operator
- Realm
- Event

## Not allowed

- Blue Team should not have cards categorized as Attack.
- Red Team should not have cards categorized as Defense.

## Examples

Correct:

- Blue Team: Basic MFA = Defense
- Blue Team: Field Technician = Operator
- Blue Team: Operations Budget = Realm
- Red Team: Port Scan = Attack
- Red Team: Password Spray = Attack
- Red Team: Alert Fatigue = Event

Incorrect:

- Blue Team: Manual Override categorized as Attack
- Red Team: Public Exploit categorized as Defense

## Design note

Red Team cards should feel like intrusion, disruption, pressure, or exploitation.

Blue Team cards should feel like detection, response, recovery, reinforcement, or resilience.
